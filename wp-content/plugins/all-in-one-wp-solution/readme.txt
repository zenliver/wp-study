== Description ==

If you manage a multi-author WordPress blog or WordPress sites for clients, then you may have wondered if it was possible to clean up the WordPress site head for you or for your users and also enable disable features of WordPress? There are lots of things in the WordPress admin area that your users don’t need to see or use. This plugin help you to remove, disable unnecessary items and or customize Wordpress site.

= What does this plugin do? =

All In One WP Solution is a powerful plugin for all wordpress user. Using this plugin, enable, disable or remove the unwanted meta tags, links and features on a WordPress site running 4.4 and beyond. This plugin helpes to customize the wordpress admin area.
This plugin helps to remove and disable WP Generator Meta, RSD Link, XML-RPC Link, Feed Link, Shortlinks and WP Json Rest API from source code and it has many more cool features and options.

== Installation ==

1. Visit 'Plugins > Add New'.
1. Search for 'All In One WP Solution' and install it.
1. Or you can upload the `all-in-one-wp-solution` folder to the `/wp-content/plugins/` directory manually.
1. Activate All In One WP Solution from your Plugins page.
1. After activation go to 'Settings > WP Solution'.
1. Enable options as per your need and save changes.

== Frequently Asked Questions ==

= How to use this plugin? =

Go to 'All In One WP Solution' from your settings menu and check the options and hit 'save changes'.

= Is there an admin interface for this plugin? =

Yes. You can access this from your admin menu. Go to 'WP Solution > Dashboard' option.

= How to check this is working? =

Open any page's source code of your website and you will see the change.

= What can I do, if my website is break due to malfunction of this plugin? =

Just deactive and reactive this plugin.

Use the plugin to test it.
